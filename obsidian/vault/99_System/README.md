# Obsidian Vault Layout

- `00_Inbox_PDFs/`: put new PDF files here.
- `00_Inbox_PDFs/_processed/`: processed PDFs are archived here automatically.
- `10_Papers/<paper_id>/`: generated paper workspace for Obsidian.

Run auto mode:

```bash
scripts/obsidian_auto_add.sh
```

Run one cycle:

```bash
scripts/obsidian_auto_add.sh --once
```
